# CV folder

Place the downloadable CV here as `Sayina_Attaf_CV.pdf`.

The Navbar, Hero, and About page all link to `/cv/Sayina_Attaf_CV.pdf` (resolved
relative to `import.meta.env.BASE_URL`, so it works on both GitHub Pages and
Vercel). Until a real file is added, the download link will 404 — replace this
folder's contents with the actual CV PDF before deploying.
